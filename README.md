# binomialOptions

<!-- badges: start -->
<!-- badges: end -->

The goal of binomialOptions is to be able to valuate binomial options
using the Cox, Ross and Rubinstein adaptation of OPTIONS, FUTURES, AND
OTHER DERIVATIVES book from John C. Hull

To run the application:

``` r
renv::restore()
rhino::app()
```
